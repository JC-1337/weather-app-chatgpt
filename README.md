# Weather App
Una aplicación web para consultar el clima actual y el pronóstico de los próximos días. El proyecto fue desarrollado en conjunto con ChatGPT para poner en práctica la conexión con una API y obtener datos de ésta.

## ¿Qué hace la app?

- Búsqueda: Al escribir una ciudad, aparece un menú para elegir la ubicación exacta.
- Datos actuales: Muestra la temperatura, el viento y si el cielo está despejado o lluvioso.
- Pronóstico: Incluye una sección con el clima previsto para los siguientes 4 días.
- Visual: La interfaz cambia el icono y la imagen de fondo dependiendo de si está soleado o lloviendo.

## Cómo usarla
1. Escribe el nombre de una ciudad en el buscador.
2. Selecciona la opción correcta en el menú que se despliega.
3. La información se actualizará automáticamente en la tarjeta principal y en el pronóstico inferior.

## Tecnologías y Recursos
- Lenguajes: HTML, CSS y JavaScript.
- Datos: Utiliza la API de Open-Meteo para la geocodificación y el clima.
- Imágenes: Las fotos de fondo fueron obtenidas de Unsplash.
